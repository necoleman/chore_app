# App icons

`icon-192.png` (192×192) and `icon-512.png` (512×512) are the PWA install icon and
the notification icon (used as `icon`/`badge` in `src/sw.js`).

**These are placeholder solid-green (`#16a34a`) squares** so the build and notifications
have a valid icon. Replace them with your real artwork at the same paths and sizes. The
512 should be **maskable** (keep important content within the center ~80% safe zone).
